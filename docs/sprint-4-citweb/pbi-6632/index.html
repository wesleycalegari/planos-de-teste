CT-ALF-001 — WS CTE: CNPJ segurado alfanumérico

Pré-condição: Endpoint WS CTE disponível em homologação. CNPJ alfanumérico de teste definido com analista.

  • Enviar para o WS CTE
  • Verificar a resposta

Resultado Esperado: WS CTE aceita o CNPJ alfanumérico sem erro de validação. Averbação processada normalmente. CNPJ persistido corretamente na base.
