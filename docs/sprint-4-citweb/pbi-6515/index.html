📝 Descrição

O CITWEB já possui o módulo Importar/Exportar com suporte à consulta de movimentos de integração para outros ramos. Esta funcionalidade precisa ser estendida ao ramo RCV (59), aproveitando a integração de averbações AT&M já existente. A solução deve ser padronizada e aplicável a todas as seguradoras, sem restrição por empresa.

🎯 Escopo

✅ Dentro do escopo

  • Acesso à tela de consulta no módulo Importar/Exportar para ramo RCV
  • Filtros: data, apólice, status de integração, seguradora
  • Listagem de movimentos com divergência CITWEB x AT&M
  • Comportamento idêntico ao padrão dos outros ramos
  • Disponibilidade para todas as seguradoras
  • Relatório exportável com lista de inconsistências
  • Validação do indicativo Ind_Existe_tb_itf_atm_ws

🚫 Fora do escopo

  • Correção automática das inconsistências
  • Alteração no fluxo de averbação AT&M
  • Outros ramos além de RCV (exceto regressão)
  • Testes de automação
