IMPOR-01 Deve exigir confirmações na conversão quando garantia, navio ou transbordo diferem
LIVE AUTO
Pré-condições (Arrange)
  • Login no sistema
  • Criar apólice de Importação com subgrupo
  • Registrar averbação provisória com dados específicos
  • Navegar para Internacional > Averbação > Importação > Conversão
Passos de Execução (Act)
  1. Selecionar a averbação provisória para conversão
  2. Informar dados definitivos com garantia, navio OU transbordo diferentes
  3. Clicar em Converter
Resultado Esperado (Assert)
  • Sistema deve exibir diálogo de confirmação para cada campo divergente
  • Conversão só deve prosseguir após confirmação de todas as divergências
  • Se o usuário cancelar qualquer confirmação, a conversão deve ser abortada
